### Instalação


1. Clone do repo
   ```sh
   git clone https://github.com/ElvisFelix-dev/frontend-tropa-digital.git
   ```
   
2. Install NPM packages
   ```sh
   cd frontend-tropa-digital
   ```
3. Instalar NPM packages
   ```sh
   npm install
   ```
   
4. Iniciar
   ```sh
   npm run dev
   ```
